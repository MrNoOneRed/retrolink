# RetroLink

> Modern open-source platform for retro game cartridges.

---

## Vision

RetroLink is an open-source hardware and software platform that aims to modernize original retro consoles while preserving the authentic gaming experience.

The first target platform is **Nintendo Entertainment System (NES)**.

Long-term goals include support for:

- SNES
- Sega Mega Drive
- Game Boy
- other retro platforms

RetroLink is **not only a flash cartridge**.

It is intended to become a modern platform featuring:

- WiFi connectivity
- OTA firmware updates
- Save synchronization
- RetroAchievements support
- Web interface
- Open SDK for homebrew developers

---

# Main goals

1. Learn electronics from scratch.
2. Learn PCB design.
3. Learn embedded programming.
4. Build a production-ready flash cartridge.
5. Open-source the entire project.

# Project philosophy

This repository is not only a product.

It is also a learning journey.

Every important engineering decision should be documented.

Knowledge is as important as code.

The project values **continuous progress** over speed.

Completing one small task every evening is preferred over attempting large, unfinished features.

A milestone is achieved through many small, successful sessions.

## ⏱ Development Sessions

This project is developed in small, focused sessions.

### Session Rules

- A single task (RL-xxx) should take **no more than 2–3 hours** to complete.
- If a task cannot be completed within 3 hours, it should be split into smaller tasks.
- Every session should end with either:
  - a working result,
  - a documented decision,
  - or a clear understanding of the explored topic.
- Avoid leaving partially completed tasks whenever possible.

The goal is to keep the project enjoyable and sustainable while balancing work, family life, and learning.
