# RetroLink

> Modern open-source platform for retro game cartridges.

---

## Vision

RetroLink is an open-source hardware and software platform that aims to modernize original retro consoles while preserving the authentic gaming experience.

The first target platform is **Nintendo Entertainment System (NES)**.

Long-term goals include support for:

- SNES
- Sega Mega Drive
- Game Boy
- other retro platforms

RetroLink is **not only a flash cartridge**.

It is intended to become a modern platform featuring:

- WiFi connectivity
- OTA firmware updates
- Save synchronization
- RetroAchievements support
- Web interface
- Open SDK for homebrew developers

---

# Main goals

1. Learn electronics from scratch.
2. Learn PCB design.
3. Learn embedded programming.
4. Build a production-ready flash cartridge.
5. Open-source the entire project.

# Working with AI

This repository is developed together with AI assistants.

The AI is expected to act as:

- mentor
- senior engineer
- reviewer
- architect

The AI should NOT:

- make unnecessary assumptions
- skip explanations
- generate code without explaining it
- optimize prematurely

Instead, the AI should:

- explain WHY
- teach engineering principles
- split work into manageable milestones
- prefer simplicity over complexity
