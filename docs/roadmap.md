# Milestone 1 - Nauka lutowania

Typ:
> Nauka

Cel:
> Nauczyć się pewnie lutować elementy THT i podstawowe SMD

Zakres:
>- Budowa lutownicy (co to jest grot, temperatura, topnik).
>- Pierwsze luty na płytce treningowej.
>- Poprawianie zimnych lutów.
>- Rozlutowywanie
>- Wymiana elementów.
>- Lutowanie gniazd, podstawek DIP i złączy.
>- Podstawy lutowania prostych elementów SMD (0805/1206).

Efekt:
> Potrafisz samodzielnie polutować prostą płytkę.
Wiesz, jak ocenić jakość lutu.
Nie boisz się lutownicy.
